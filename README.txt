# JARVIS HUD WebApp

실제 자비스 화면처럼 보이도록 만든 웹앱 버전입니다.

## 업로드 파일
- index.html
- style.css
- app.js
- manifest.json
- README.txt

## GitHub Pages 사용
기존 jarvis 저장소에 위 파일들을 덮어쓰기 업로드하세요.
Settings → Pages → main / root 설정이면 주소 하나로 노트북과 핸드폰에서 사용할 수 있습니다.

## 기능
- 자비스 HUD 화면
- 중앙 코어 버튼 음성 명령
- 텍스트 명령
- 메모 저장/조회/삭제
- 간단 계산
- 노트북/핸드폰 반응형 화면
