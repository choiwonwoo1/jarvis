# Jarvis WebApp Final

## 사용 방식
이 파일들을 GitHub Pages에 업로드하면 노트북과 핸드폰에서 같은 주소로 사용할 수 있습니다.

## 업로드할 파일
- index.html
- style.css
- app.js
- manifest.json
- README.txt

## GitHub Pages 설정
1. GitHub 저장소 jarvis 열기
2. 기존 파일 삭제하거나 덮어쓰기
3. 위 파일 업로드
4. Settings → Pages
5. Source: Deploy from a branch
6. Branch: main / root
7. Save
8. 주소 접속

## 핸드폰 앱처럼 사용
Android Chrome:
1. GitHub Pages 주소 접속
2. 오른쪽 위 점 3개
3. 홈 화면에 추가

## 기능
- 노트북/핸드폰 반응형 화면
- 음성 인식
- 음성 답변
- 메모 저장
- 메모 목록
- 간단 계산
