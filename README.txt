# 모바일 Jarvis v2

## 변경 내용
- 핸드폰 화면에 맞게 모바일 전용 UI로 변경
- 전기공사 빠른 계산 삭제
- 등/스위치 배선 계산 삭제
- 메모, 간단 계산, 음성 명령 중심으로 정리

## 실행 방법
1. 압축을 풉니다.
2. index.html 파일을 핸드폰으로 옮깁니다.
3. Android는 Chrome에서 열면 가장 잘 작동합니다.

## 핸드폰에서 쉽게 쓰는 방법
가장 편한 방법은 파일을 웹호스팅이나 GitHub Pages에 올린 뒤 주소로 접속하는 방식입니다.
그 다음 Chrome 메뉴에서 "홈 화면에 추가"를 누르면 앱처럼 실행할 수 있습니다.

## 예시 명령
- 자비스, 메모 태양광 철거 견적 확인
- 자비스, 메모 보여줘
- 자비스, 1502 나누기 30 계산해줘
- 자비스, 안녕

## 주의
iPhone Safari는 브라우저 음성 인식 기능이 제한될 수 있습니다.
Android Chrome 사용을 권장합니다.
