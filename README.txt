# JARVIS HUD News WebApp

## 추가된 기능
- “자비스, 주요 뉴스 알려줘”
- “자비스, 전기 뉴스 알려줘”
- 뉴스 패널 표시
- 뉴스 제목 클릭 시 원문 열기

## 업로드 방법
기존 GitHub jarvis 저장소에 아래 파일을 덮어쓰기 업로드하세요.

- index.html
- style.css
- app.js
- manifest.json
- README.txt

## 사용 예시
- 자비스, 주요 뉴스 알려줘
- 자비스, 전기 뉴스 알려줘
- 자비스, 메모 보여줘
- 자비스, 1502 나누기 30 계산해줘

## 참고
뉴스는 Google News RSS를 공개 프록시로 읽는 방식입니다.
무료 방식이라 접속 환경에 따라 잠시 불러오지 못할 수 있습니다.
나중에 더 안정적으로 만들려면 별도 서버 또는 뉴스 API를 연결하면 됩니다.
